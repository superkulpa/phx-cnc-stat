Порядок проведение тестов:

Тест 1:
- задать необходимые даты и нажать кнопку "Отчет".
- для сохранения отчета - кнопка "Сохранить"

Тест 2:
- настроить FTP-подключение.
- настроить путь к скрипту создания архива.
- заполнить поле с именем выходного архива на сервере.
- заполнить путь к архиватору с параметрами его запуска. Можно использовать переменные:
	- [ARCHIVE_NAME] - имя архива на стороне сервера (оно же будет у загруженного архива).
	- [APP_PATH] - абсолютный путь до папки программы.
	- Пример: "C:\Program Files (x86)\WinRAR\rar.exe" e -o+ "[ARCHIVE_NAME]" "[APP_PATH]/logs"

- Нажать кнопку "Получить по сети". После успешного скачивания, архив будет автоматически распакован и появится сообщение "Архив успешно скачан и распакован" иначе сообщение "Не удалось разархивировать архив:" с именем архива.
- Можно также указать архив на компьютере с помощью кнопки "Получить из архива"
- После распаковки архива необходимо проверить наличие данных в папке logs, расположенной рядом с программой (при условии правильной настройки пути к архиватору!).
